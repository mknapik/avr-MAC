MAC API Callback Functions (Confirm and Indication Functions)

Detailed Description

This group of functions defines the interface for messages from the stack to the application implementing the confirmation and indications required by IEEE 802.15.4.

For each event, which the MAC has to signal to the application layer, one of the functions of this group is called. The stack contains empty implementations for each function, which can be overwritten by the application.
