PAL Example Application - Simple LED Remote Control

Abstract
The PAL (Platform Abstraction Layer) Example Application "Simple LED Remote Control" is an initial application for the ATmega128RFA1 device. It demonstrates how to send and receive frames using PAL API Functions. The example application uses the Radio Controller Board (RCB) as the hardware platform. Once the board is powered, the application starts automatically. If a button is pressed a 802.15.4 data frame is transmitted. When a frame is received, its address field and CRC are checked and if they match, the RX counter is incremented and displayed using the board's LEDs.

This example application can be used for simple range tests.
